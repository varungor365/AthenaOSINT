## üéØ Quick Start Examples

### Example 1: Monitor Your Company
```bash
# SSH to droplet
ssh root@143.110.254.40

# Start harvester service
systemctl start athena-harvester

# Add your company domain
curl -X POST http://localhost:5000/api/harvester/add-target \
  -H "Content-Type: application/json" \
  -d '{"type":"domain","target":"yourcompany.com"}'

# Add CEO email
curl -X POST http://localhost:5000/api/harvester/add-target \
  -H "Content-Type: application/json" \
  -d '{"type":"email","target":"ceo@yourcompany.com"}'

# Check alerts
curl http://localhost:5000/api/harvester/alerts
```

### Example 2: Leak Monitoring
```bash
# Add sensitive keywords
curl -X POST http://localhost:5000/api/harvester/add-target \
  -d '{"type":"keyword","target":"yourcompany confidential"}'

curl -X POST http://localhost:5000/api/harvester/add-target \
  -d '{"type":"keyword","target":"yourcompany API key"}'

# Harvester will scan GitHub, Pastebin, Google for these keywords
```

### Example 3: Continuous Vuln Scanning
```bash
# Add all your domains/subdomains
for domain in web.company.com api.company.com admin.company.com; do
  curl -X POST http://localhost:5000/api/harvester/add-target \
    -d "{\"type\":\"domain\",\"target\":\"$domain\"}"
done

# Harvester will run Nuclei scans every 30 minutes
```
